# 3dprintdesigns

A collection of 3D print and CAD designs organised by category. Python macro files may also be included within each category to support parametric or automated design workflows.

## Categories

- **[music/](music/)** – Designs related to musical instruments, accessories, stands, and storage.
- **[home/](home/)** – Household items, organisers, fixtures, and general home utility prints.
- **[wargaming/](wargaming/)** – Miniatures, terrain, tokens, and accessories for tabletop wargames.
- **[kids/](kids/)** – Toys, educational models, and child-friendly prints.

## File Types

| Extension | Description |
|-----------|-------------|
| `.stl` | Standard mesh file for slicing and printing |
| `.3mf` | 3D Manufacturing Format – richer alternative to STL |
| `.step` / `.stp` | STEP CAD file – editable solid model |
| `.f3d` | Fusion 360 source file |
| `.scad` | OpenSCAD parametric source file |
| `.py` | Python macro / script (e.g. FreeCAD macros, parametric helpers) |
