# Home

3D print designs for around the house — organisers, fixtures, repairs, and general utility items.

## Contents

Add designs here. Suggested sub-folders as the collection grows:

- `kitchen/` – utensil holders, jar openers, hooks
- `bathroom/` – toothbrush holders, soap dishes
- `office/` – cable clips, monitor stands, desk organisers
- `garden/` – plant labels, tool holders

Python macro files (`.py`) for parametric designs can also be placed here.
