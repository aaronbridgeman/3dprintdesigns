# Kids

3D print designs for children — toys, puzzles, educational models, and fun prints.

## Contents

Add designs here. Suggested sub-folders as the collection grows:

- `toys/` – cars, figures, play accessories
- `puzzles/` – interlocking puzzles, shape sorters
- `educational/` – letters, numbers, science models
- `outdoor/` – garden games, chalk holders

Python macro files (`.py`) for parametric designs can also be placed here.
