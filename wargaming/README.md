# Wargaming

3D print designs for tabletop wargames — miniatures, terrain pieces, tokens, trays, and accessories.

## Contents

Add designs here. Suggested sub-folders as the collection grows:

- `terrain/` – buildings, walls, trees, scatter terrain
- `miniatures/` – figures, mounts, vehicles
- `accessories/` – dice towers, movement trays, tokens, markers
- `storage/` – carry cases, foam inserts, magnetised trays

Python macro files (`.py`) for parametric designs can also be placed here.
